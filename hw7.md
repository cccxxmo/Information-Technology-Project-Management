## 3. 假设项目已经做了3个月，而这个为期6个月的项目的BAC是200000美元。
- PV = 120000 美元
- EV = 100000 美元
- AC = 90000 美元
### a. 这个项目的成本偏差、进度偏差、成本绩效指数（CPI）和进度绩效指数（SPI）是多少？
- CV（成本偏差） = EV-AC = 100000-90000 = 10000 美元
- SV（进度偏差） = EV-PV = 100000-120000 = -20000 美元
- CPI（成本绩效指数） = EV/AC = 100000/90000 = 111%
- SPI（进度绩效指数） = EV/PV = 100000/120000 = 83.3%
### b. 这个项目的运作情况如何？它是提前于进度还是落后于进度？它是在预算中还是超出了预算？
进度偏差是负的，说明项目的运作落后于进度。成本绩效指数大于1，说明项目到此没有超出预算。
### c. 使用CPI来计算这个项目的完工估计（EAC）。这个项目比计划表现得好还是差？
EAC = BAC/CPI = 200000/111% = 180000 美元   
从进度偏差和成本偏差可以看出，这个项目在进度上比计划表现得差，但在成本上比计划表现的好。
### d. 使用进度绩效指数（SPI）来估计将花费多长时间来完成这个项目。
6月/SPI ≈ 7.2月   
故估计将花费4~5个月来完成这个项目。
